
# MediSmart – AI-Powered Prescription Analyzer

## Overview

MediSmart is an AI-powered mobile healthcare application that helps users understand prescriptions by identifying medicines from uploaded prescriptions and suggesting relevant information and alternatives.

## Features

* Prescription image upload
* AI-based medicine identification
* Alternative medicine recommendations
* Medicine information lookup
* User-friendly Flutter interface
* Flask API integration
* Machine Learning prediction model

## Technologies Used

* Flutter
* Python
* Flask API
* Machine Learning
* Pandas
* Scikit-learn
* GitHub

## System Architecture

1. User uploads prescription image.
2. OCR extracts medicine names.
3. Flask API processes the request.
4. Machine Learning model predicts medicine information.
5. Results are displayed in the Flutter application.

## Installation

### Flutter App

```bash
flutter pub get
flutter run
```

### Flask Backend

```bash
pip install -r requirements.txt
python app.py
```
